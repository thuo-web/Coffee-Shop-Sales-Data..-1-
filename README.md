# Coffee-Shop-Sales-Data..-1-

Data analysis project exploring coffee shop sales trends using Excel and data visualization techniques. Includes dashboards, insights, and KPI tracking.


##  Project Overview

This project focuses on analyzing coffee shop sales data to identify revenue trends, top-performing products, customer behavior patterns, and location performance. The analysis transforms raw sales data into meaningful visual insights to support business decision-making.


##  Objectives

- Analyze overall sales performance
- Identify top-selling products and categories
- Understand revenue distribution across cities
- Evaluate sales trends by weekday
- Create an interactive dashboard for data-driven insights


##  Tools Used

- Microsoft Excel
- Data Cleaning and Transformation
- Pivot Tables
- Data Visualization
- Dashboard Design
